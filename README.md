# 🦈 Shark Camps — Sistema Web para Academia de Boxe

Sistema web completo desenvolvido do zero para uma academia de boxe, como primeiro projeto real de portfólio.

## 🚀 Tecnologias utilizadas

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express.js
- **Banco de dados:** MySQL
- **Autenticação:** JWT + bcrypt
- **Email:** Nodemailer + Gmail
- **Versionamento:** Git + GitHub

## ⚙️ Funcionalidades

- ✅ Tela de login com autenticação real (JWT)
- ✅ Dashboard com informações da academia
- ✅ Sistema de planos com termos e condições
- ✅ Loja online (Shark Store) com seleção de cor e tamanho
- ✅ Pedidos salvos no banco de dados MySQL
- ✅ Notificação automática por email a cada novo pedido
- ✅ Middleware de proteção de rotas
- ✅ API REST completa

## 🗄️ Banco de dados

Tabelas criadas:
- `users` — usuários e autenticação
- `plans` — planos da academia
- `students` — alunos matriculados
- `orders` — pedidos da loja
